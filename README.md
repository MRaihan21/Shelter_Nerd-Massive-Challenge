## What is XET
XET merupakan aplikasi mobile (Android) yang dirancang khusus untuk mempermudah pencarian dan pembelian tiket event dengan berbagai genre untuk para penggemar event. Aplikasi ini juga menyediakan berbagai fitur untuk memudahkan pengguna seperti, Pengguna dapat mencari event berdasarkan genre, lokasi, tanggal, dan lainnya.
